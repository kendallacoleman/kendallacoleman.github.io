JM'Uvies Website

"JM'Uvies" is a movie social media website that my team and I created for CS345 (Software Engineering).


My team managed project milestones and deliverables using the Agile Scrum Framework. We created an interactive movie review system, allowing users to submit reviews, search for any movie they want, and see other profile's reviews. This project was developed over the course of 6 weeks using JavaScript, HTML, and CSS. We also used "The Movie Database" API for fetching and displaying data.


In this project, my main focus was developing the front-end design of the profile page, the posts themselves, and the layout of the movie information pages. This project allowed me to explore my interest in front-end development for the first time and get a strong understanding of javascript.
