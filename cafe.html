Cafe Web Application

"Sunny Side Cafe" is a fictional website that my team and I created in CS343 (Application Development).


We created a user-friendly and visually appealing website for "Sunny Side Cafe" using Javascript, HTML, and CSS. The website also implements an online ordering system that we designed, alongside integrated location-based services using the Google Maps API to enhance user experience.


In this project, my main focus was developing the CSS for the mobile view of the website. This allowed me to get a deeper understanding of CSS and how it works!
